<회원가입>
1. 시스템은 ID, 비밀번호, 이름, 전화번호, 이메일 입력 항목을 표시한다.
2. 사용자는 ID, 비밀번호, 이름, 전화번호, 이메일을 입력하고 회원 가입을 요청한다.
3. 시스템은 입력 정보가 회원 가입 조건을 만족하는지 확인하고, 회원 가입 승인 팝업을 표시한다.
4. 사용자는 회원 가입 승인 팝업을 확인한다. 
5. 시스템은 로그인 화면을 표시한다.

<로그인>
1. 시스템은 ID와 비밀번호 입력을 표시한다.
2. 회원 또는 관리자는 ID와 비밀번호를 입력하고 로그인을 요청한다.
3. 시스템은 입력된 ID와 비밀번호가 로그인 조건을 만족하는지 확인하고, 로그인 처리 후 회원용 또는 관리자용 메인 화면을 표시한다. 

<회원용 메인 화면 조회, 로그아웃, 회원 탈퇴>
1. 시스템은 회원용 메인 화면에 로그아웃 버튼과 회원 탈퇴 버튼을 표시한다.

Extensions
1번에서 회원이 로그아웃 버튼을 선택하면, 시스템은 회원을 로그아웃 처리하고 로그인 화면을 표시한다.
1번에서 회원이 회원 탈퇴를 선택하면, 시스템은 회원 탈퇴를 처리하고 회원 탈퇴 완료 결과를 표시한다. 이때 해당 회원의 이용 권한과 관련 데이터는 삭제된다.

<관리자용 메인 화면 조회, 로그아웃>
1. 시스템은 관리자용 메인 화면에 로그아웃 버튼을 표시한다.
2. 관리자가 로그아웃 버튼을 선택한다.
3. 시스템은 관리자를 로그아웃 처리하고 로그인 화면을 표시한다.
