<회원가입>
1. 시스템은 ID, 비밀번호, 이름, 전화번호, 이메일 입력 항목을 표시한다.
2. 사용자는 ID, 비밀번호, 이름, 전화번호, 이메일을 입력하고 회원 가입을 버튼을 클릭한다.
3. 시스템은 회원 가입 승인 팝업을 표시한다.

<로그인>
1. 시스템은 ID와 비밀번호 입력을 표시한다.
2. 회원 또는 관리자는 ID와 비밀번호를 입력하고 로그인 버튼을 클릭한다.
3. 시스템은 회원용 또는 관리자용 메인 화면을 표시한다. 

<회원용 메인 화면 조회, 로그아웃, 회원 탈퇴>
1. 시스템은 회원용 메인 화면에 로그아웃 버튼과 회원 탈퇴 버튼을 표시한다.

Extensions

1단계 이후,
1. 회원이 로그아웃 버튼을 클릭한다.
2. 시스템은 로그아웃이 완료되었다는 메시지를 표시한다.

1단계 이후,
1. 회원이 회원 탈퇴 버튼을 클릭한다.
2. 시스템은 재확인을 위한 메시지 창을 띄운다.
3. 회원이 확인 버튼을 클릭한다.
4. 시스템은 회원 탈퇴가 완료되었다는 메시지를 표시한다.

<관리자용 메인 화면 조회, 로그아웃>
1. 시스템은 관리자용 메인 화면에 로그아웃 버튼을 표시한다.

Extensions

1단계 이후,
1. 관리자가 로그아웃 버튼을 클릭한다.
2. 시스템은 로그아웃이 완료되었다는 메시지를 표시한다.
