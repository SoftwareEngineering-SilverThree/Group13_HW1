* 설문 검색

1. 회원은 설문 검색 화면에 접속한다.
2. 시스템은 설문 검색 화면과 검색 조건 입력 항목을 표시하고 조회할 설문 상태를 표시한다.
3. 회원은 조회할 설문 상태를 선택하고 검색 키워드를 입력한 후 검색 버튼을 클릭한다.
4. 시스템은 회원이 입력한 검색 조건을 확인한 후 조건에 맞는 설문 데이터를 조회하여 검색 결과에 해당하는 설문 목록을 화면에 출력한다.
5. 회원은 출력된 설문 목록을 확인한다.

Alternative courses
검색 결과가 없는 경우“검색 결과가 없습니다”메시지를 출력한다.



* 회원 설문 상세정보조회

1. 회원은 설문 목록 화면을 확인한다.
2. 회원은 조회할 설문을 선택한다.
3. 시스템은 선택된 설문 정보를 확인하고 해당 설문의 상세 정보를 조회한다.
4. 시스템은 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각을 화면에 출력한다.
5. 회원은 출력된 설문 상세 정보를 확인한다.
6. 시스템은 설문 진행 상태에 따라 응답 가능 여부를 표시하고, 현재 진행 중인 설문인 경우에만 응답 버튼을 활성화한다. 
6-1. 진행 예정이거나 또는 종료된 설문인 경우 시스템은 응답 버튼을 비활성화한다.
