<설문 응답>
1. 회원은 현재 진행중인 설문조사 중 하나를 선택한다.
2. 시스템은 선택한 설문조사의 설문항목이 있는 응답 화면을 제공한다
3. 회원은 설문조사 응답 항목을 모두 입력하고 ‘응답’ 버튼을 눌러 완료한다.

<응답한 설문 조회, 설문 응답 취소>
1. 시스템은 회원이 응답한 설문 리스트를 제공한다.
1-1. 현재 진행중인 설문조사라면 수정 버튼과 취소 버튼이 활성화된다. 설문이 종료되었다면 수정버튼과 취소버튼을 비활성화한다.
2. 회원이 응답한 설문 리스트 중에 하나를 선택한다.
3. 시스템은 선택된 설문조사의 응답 내용을 팝업으로 화면에 출력한다.

Extensions
1단계 이후, 회원이 현재 진행중인 설문 중에서 응답 취소를 하고 싶다면, 설문의 ‘취소’ 버튼을 누른다.
이후 시스템에서 해당 설문을 응답한 설문 리스트와 조회 화면에서 제거한다. 

<응답한 설문 수정>
1. 시스템은 회원이 응답한 설문조사 리스트를 제공한다.
2. 회원은 현재 진행중인 응답한 설문조사 중 수정을 원하는 설문조사의 ‘수정’ 버튼을 누른다.
3. 시스템은 수정화면에서 설문조사에 대한 설문항목을 제공한다. 이때 최근 수정 날짜와 이전에 응답한 내용을 출력한다.
4. 회원은 각 설문 항목 중 원하는 항목을 수정한다. 수정 버튼을 누른다. 
5. 시스템에서 응답한 설문 조회화면으로 돌아간다.
