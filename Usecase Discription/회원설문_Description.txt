<설문 응답>
1. 시스템은 설문 제목, 설문 설명, 설문 문항, 응답 항목을 화면에 표시한다.
2. 회원은 설문조사 응답 항목을 모두 입력하고 ‘응답’ 버튼을 눌러 완료한다.
3. 시스템은 응답이 완료되었다는 메시지를 표시한다.


<응답한 설문 리스트 조회, 응답한 설문 상세정보 조회, 설문 응답 취소>
1. 시스템은 회원이 응답한 설문 리스트를 표시한다. 현재 진행중인 설문조사라면 수정 버튼과 취소 버튼이 활성화된다.

Extensions

1단계 이후,
1. 회원은 리스트에서 응답 내용을 확인하고자 하는 항목을 선택한다.
2. 시스템은 선택된 설문의 제목, 응답 내용, 응답 일시를 팝업으로 화면에 출력한다.

1단계 이후,
1. 회원이 현재 진행 중인 설문의 ‘취소’ 버튼을 누른다.
2. 시스템은 삭제 재확인을 위한 메시지 창을 띄운다.
3. 회원이 삭제 확인 버튼을 누른다.
4. 시스템은 해당 설문이 제거된 리스트를 화면에 출력한다.


<응답한 설문 수정>
1. 시스템은 설문 제목, 회원이 이전에 답변한 응답 내용, 가장 최근에 수정한 날짜를 출력한다.
2. 회원은 각 설문 항목 중 원하는 항목을 수정하고 수정 버튼을 누른다.
3. 시스템에서 수정이 완료되었다는 메시지를 표시한다.
