# 회원가입, 탈퇴, 로그인/로그아웃 

Use Case - 회원 가입
1. 시스템을 이용하려는 사용자는 회원 가입을 할 수 있어야 한다. 회원 가입 시 사용자는 ID, 비밀번호, 이름, 전화번호, 이메일을 필수로 입력해야 한다.

Use Case - 관리자 로그인
2. 관리자는 주어진 ID(admin)와 비밀번호(admin)를 사용해 로그인할 수 있어야 한다.

Use Case - 회원 로그인
3. 회원은 ID와 비밀번호를 사용해 로그인할 수 있어야 한다.

Use Case - 로그아웃
4. 관리자와 회원은 로그인한 상태에서 로그아웃할 수 있어야 한다.

Use Case - 회원 탈퇴
5. 회원은 언제든지 회원 탈퇴를 할 수 있어야 한다. 
6. 회원 탈퇴 시 해당 회원의 모든 이용 권한과 데이터가 삭제되어야 한다.
