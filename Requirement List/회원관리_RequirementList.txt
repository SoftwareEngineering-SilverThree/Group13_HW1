# 최종_진짜_진짜진짜_0123

No. 1
Requirement - 시스템을 이용하려는 비회원은 ID, 비밀번호, 이름, 전화번호, 이메일을 입력하여 회원 가입을 할 수 있어야 한다.
Use Case - 회원 가입

No. 2
Requirement - 회원과 관리자는 ID와 비밀번호를 입력하여 로그인할 수 있어야 한다. 이때 관리자는 주어진 ID(admin)와 비밀번호(admin)를 사용하여 로그인할 수 있어야 한다.
Use Case - 로그인

No. 3
Requirement - 회원은 로그인한 상태에서 회원 설정 화면을 조회할 수 있어야 한다. 이때 회원은 회원 설정 화면에서 로그아웃할 수 있어야 하며, 회원 탈퇴를 할 수 있어야 한다. 회원 탈퇴 시 해당 회원의 모든 이용 권한과 데이터가 삭제되어야 한다.
Use Case - 회원 설정 화면 조회, 로그아웃, 회원 탈퇴

No. 4
Requirement - 관리자는 로그인한 상태에서 관리자 설정 화면을 조회할 수 있어야 한다. 이때 관리자는 관리자 설정 화면에서 로그아웃할 수 있어야 한다.
Use Case - 관리자 설정 화면 조회, 로그아웃
