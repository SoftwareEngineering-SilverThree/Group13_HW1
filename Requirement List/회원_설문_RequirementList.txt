# 응답하기, 설문 조회, 수정 및 취소 기능
Use case : 설문조사 응답하기
1. 회원은 응답 화면에서 설문에 응답을 완료할 수 있어야 한다.

Use case : 응답한 설문 조회
2. 회원은 응답한 모든 설문 정보를 한 화면에서 조회할 수 있어야 한다. 각 응답한 설문에 대해서 설문 제목, 응답 내용, 응답 일시 등이 출력된다. 

Use case : 응답한 설문 취소
3. 회원은 응답한 설문 조회 화면에서 현재 진행 중인 설문에 대해서 취소를 할 수 있어야 한다.

Use case : 응답한 설문 수정
4. 회원은 응답한 설문 조회 화면에서 현재 진행 중인 설문에 대해서 수정을 할 수 있어야 한다.
