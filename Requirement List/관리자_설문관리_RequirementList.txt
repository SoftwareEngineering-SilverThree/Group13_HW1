설문 등록/조회/삭제 기능, 통계 기능

Use Case - 설문 리스트 조회
1. 관리자가 설문 리스트 조회 화면에서 등록된 모든 설문 목록을 조회할 수 있어야 한다.

Use Case - 설문 등록
2. 관리자가 설문 리스트 조회 화면에서 설문 등록을 선택하면 팝업 창으로 설문을 등록할 수 있어야 한다. 설문 등록 시 입력 정보에는 설문 제목, 설문 설명, 설문 문항, 응답 항목, 설문 시작 시각, 설문 마감 시각이 포함된다.

Use Case - 설문 상세 조회
3. 관리자가 설문 리스트 조회 화면에서 특정 설문을 선택하면 해당 설문의 상세 내용을 팝업 창으로 확인할 수 있어야 한다.

Use Case - 설문 삭제
4. 관리자가 설문 리스트 조회 화면에서 특정 설문을 선택하여 삭제할 수 있어야 한다.

Use Case - 설문 통계 조회
5. 관리자는 설문 응답 통계 정보를 조회할 수 있어야 한다. 통계 조회 시 검색 단위를 최근 1주일, 1개월, 1년 중에서 선택할 수 있으며, 선택된 기간 동안의 전체 설문 수, 종료된 설문 수, 총 응답 수가 출력된다.
